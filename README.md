# CONSUTOR-DE-CADASTROS-DE-NUMERO-FeKe-
Este programa é apenas uma simulação fictícia de um sistema ilegal de consulta de dados. Ele demonstra, de forma ilustrativa, como seria um serviço que buscasse informações como CPF ou CNPJ vinculados a um número de telefone. Qualquer ferramenta real com esse propósito é proibida pela Lei Geral de Proteção de Dados (LGPD – Lei nº 13.709/2018)
